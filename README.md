# go-chess
Golang chess engine, written in order to practice Golang.

## Board struct
- A board object
- Game variables
- castle possibilities
- score
- to move
- move count
- fen string
- pgn string


## Bitmap
- 120 bytes
- 11 - 18 1st rank
- 81 - 88 8th rank
